# @design-system/tokens

Design tokens for the design system. This package contains foundational design
decisions for colors, typography, spacing, and more.

## Installation

```bash
pnpm add @design-system/tokens
```

## Usage

```typescript
import { tokens } from '@design-system/tokens';
import '@design-system/tokens/tokens.css';
```

## Development

This package is currently in development as part of Phase 2.

### Build

```bash
pnpm build
```

### Development Mode

```bash
pnpm dev
```
